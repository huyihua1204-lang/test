一、PLC 控制器故障处理
----------------------------------------------------

【设备编号】PLC-01 / PLC-02 / PLC-03
【设备类型】可编程逻辑控制器

1.1 温度传感器故障（Temperature sensor fault）
  症状：温度读数异常升高（超过 100°C）或传感器返回错误值
  原因：
    - 传感器探头老化或接触不良
    - 散热模块堵塞，环境温度过高
    - 传感器供电异常
  处理步骤：
    1. 立即检查 PLC 控制柜散热风扇是否正常运转
    2. 用红外测温仪测量控制柜实际温度，排除环境温度干扰
    3. 检查温度传感器接线端子是否松动或氧化
    4. 若读数持续异常，更换同型号传感器（备件编号：TS-PT100-A）
    5. 更换后需重新校准，设定正常工作阈值：60~85°C
  预防措施：每季度清洁散热滤网，每年更换传感器校准一次

1.2 过热检测与加热单元停机（Overheat detected, shutting down heater unit）
  症状：系统自动停止加热单元，产线暂停
  原因：温度超出安全阈值（>100°C），系统触发保护机制
  处理步骤：
    1. 不要立即强制重启，等待设备自然冷却至 80°C 以下
    2. 检查加热单元冷却水路是否畅通，流量是否正常
    3. 确认散热风扇、水泵运行状态
    4. 温度恢复正常后，手动确认复位后方可重启
  注意：严禁在温度未降低的情况下强制重启，可能损坏电路板

1.3 多传感器同时故障（Multiple sensor failures detected）
  症状：多个传感器同时离线或报错
  原因：
    - 供电总线故障或电压不稳
    - 通信总线（如 Profibus/Modbus）中断
    - 雷击或静电干扰
  处理步骤：
    1. 检查传感器供电总线电压（正常范围：24V ±5%）
    2. 用网络诊断工具检查通信总线连通性
    3. 重启通信模块后观察传感器是否恢复
    4. 若仍无法恢复，逐一排查各传感器接线
    5. 记录故障时间段，分析是否存在外部干扰（如附近大功率设备启停）


二、电机故障处理
----------------------------------------------------

【设备编号】MOTOR-01
【设备类型】三相异步电机（驱动单元）

2.1 电机堵转/急停（Motor stall detected, emergency stop triggered）
  症状：电机电流突增，转速降为零，系统触发急停
  原因：
    - 机械卡死：传动轴、齿轮、联轴器异物卡入
    - 负载突变：工件堆积或夹具动作异常
    - 变频器过载保护动作
  处理步骤：
    1. 确认急停状态后，切断电机主电源
    2. 人工检查传动机构，清除异物或卡点
    3. 手动盘车确认转动无阻力
    4. 检查变频器故障代码，复位后重新启动
    5. 首次重启以低速（额定转速 20%）运行 2 分钟确认正常后提速
  注意：急停后禁止在未排查原因的情况下直接复位重启

2.2 电机重启失败（Motor restart failed）
  症状：发出重启指令后电机未响应或再次报错停止
  原因：
    - 变频器温度过高，热保护未解除
    - 机械故障未完全排除
    - 控制回路接触器粘连或线圈损坏
  处理步骤：
    1. 等待变频器冷却（通常需 5~10 分钟）
    2. 检查变频器显示的具体故障代码（参见变频器手册附录A）
    3. 检查主接触器触点状态，必要时更换
    4. 确认急停按钮已物理复位（非仅软件复位）
    5. 按操作规程重新上电启动
  紧急联系：电机维修班组 内线 8021


三、传送带故障处理
----------------------------------------------------

【设备编号】CONVEYOR-01 / CONVEYOR-02
【设备类型】皮带式传送线

3.1 皮带张力低（Belt tension low）
  症状：张力传感器值低于 0.80（正常范围：0.85~1.10）
  原因：
    - 皮带长期使用后延伸松弛
    - 张紧轮螺栓松动
    - 皮带局部破损
  处理步骤：
    1. 停线后检查张紧轮位置，适量旋紧调节螺栓
    2. 沿皮带全长检查是否有裂纹、分层或破洞
    3. 调整后重新启动，观察张力值是否回到正常范围
    4. 若调整两次后仍偏低，申请更换皮带（备件编号：BELT-W600-L30）
  预防措施：每月检查一次皮带张力和磨损情况

3.2 皮带卡死/产线停机（Belt jam detected, line halted）
  症状：传送带停止运动，电机电流过载报警
  原因：
    - 工件卡入皮带与导槽之间
    - 皮带跑偏后与机架摩擦卡死
    - 皮带接头开裂断裂
  处理步骤：
    1. 立即按下就近急停按钮，不要强行反转驱动
    2. 清除卡入的工件或异物，注意手部安全
    3. 检查皮带跑偏情况，调整两侧导辊水平度
    4. 检查皮带接头钢扣是否完好
    5. 人工拖动皮带确认运动顺畅后方可重启
    6. 重启后低速运行 3 分钟观察是否再次卡带
  注意：严禁在皮带运动时伸手清理异物


四、传感器故障处理
----------------------------------------------------

【设备编号】SENSOR-03 / SENSOR-04 / SENSOR-05
【设备类型】湿度传感器 / 视觉传感器 / 质检传感器

4.1 传感器电池低电量（Low battery on humidity sensor）
  处理步骤：
    1. 更换指定型号电池（CR2032 或设备标签所示型号）
    2. 更换后按复位键重新初始化传感器
    3. 观察 5 分钟确认读数稳定后记录维护日志

4.2 视觉传感器离线（Vision sensor offline）
  症状：SENSOR-04 无信号，质检无法进行
  原因：
    - 网线/光纤接头松动
    - 传感器镜头过热或污染
    - 工控机视觉软件崩溃
  处理步骤：
    1. 检查传感器状态指示灯（绿色=正常，红色/熄灭=异常）
    2. 重新插拔网线接头，检查接口是否有针脚弯曲
    3. 在工控机上重启视觉采集软件（桌面快捷方式：VisionCapture）
    4. 清洁传感器镜头（使用专用镜头布，禁止使用普通纸巾）
    5. 若仍离线，重启传感器电源（断电等待 30 秒后重新上电）

4.3 质检传感器校准丢失（Calibration lost, readings unreliable）
  症状：产品尺寸检测结果持续偏差，误判率升高
  原因：
    - 传感器受震动影响，基准点偏移
    - 环境光线变化影响成像
    - 软件校准文件损坏
  处理步骤：
    1. 取出随机附带的标准校准块（存放于工具柜C-3）
    2. 打开校准程序（路径：D:\Vision\Calibration\calibrate.exe）
    3. 按屏幕提示完成九点标定校准
    4. 校准完成后用已知尺寸工件验证精度（误差应 ≤ 0.02mm）
    5. 保存校准文件并记录校准时间


五、常见错误代码速查表
----------------------------------------------------

  E001 - 温度超限      → 检查散热，参考 1.1 节
  E002 - 电机过载      → 检查机械卡阻，参考 2.1 节
  E003 - 通信超时      → 检查总线接线，参考 1.3 节
  E004 - 传感器离线    → 检查供电和接线，参考第四章
  E005 - 急停触发      → 排查原因后手动复位，参考 2.1 节
  E006 - 皮带张力异常  → 检查张紧轮，参考 3.1 节
  E007 - 校准失效      → 重新校准，参考 4.3 节
  W001 - 电池低电量    → 及时更换，参考 4.1 节
